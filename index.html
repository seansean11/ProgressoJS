Progresso.js

A lightweight, accurate jQuery plugin for preloading assets and displaying progress.

download github

Usage

  1. Download the ProgressoJS progresso.min.js plugin file.

  2. Load jQuery and include ProgressoJS plugin file

    To use ProgressoJS, youâ€™ll need to make sure both the ProgressoJS and jQuery scripts are included in the head of your HTML.

    <!-- Include jQuery 1.7+ -->
<script src="https://code.jquery.com/jquery-1.11.3.min.js">

<!-- Include ProgressoJS plugin -->
<script src="assets/progresso.min.js">

  3. Add ProgressoJS class and attributes to preload elements.

    In order to declare an element for preloading, you must add the js-progresso class. For all assets (aside from CSS background images) you must then declare the file to preload using the data-src attribute. For CSS background images you must declare the file using the data-src-css attribute. Note that you do not need to declare the elements real src attribute

    <!-- Preloaded Video -->
<video controls>
  <source class="js-progresso" data-src="video/example.mp4" type="video/mp4">
  <source class="js-progresso" data-src="video/example2.ogv" type="video/ogg">
</video>

<!-- Preloaded Image -->
<img class="js-progresso" data-src="image/example.jpg">

<!-- Preloaded Audio -->
<audio controls>
  <source class="js-progresso" data-src="audio/example.ogg" type="audio/ogg">
  <source class="js-progresso" data-src="audio/example2.mp3" type="audio/mpeg">
</audio>

<!-- Preloaded Picture -->
<picture>
   <source class="js-progresso" data-src="image/example.jpg">
   <source class="js-progresso" data-src="image/example2.jpg">
</picture>

<!-- Preloaded CSS Background Image -->
<div class="js-progresso" data-src-css="image/example.jpg">

  4. Leverage percentage for animations

    By utilizing the onProgress callback you can animate css or display your progress percentage on the screen. You can add your custom script right before the closing body tag. Click on "DEMO" to see it in action.

    <script>
$("body").progresso({
  onProgress: function(prog) {
    var percent = Math.round(prog * 100),
        percentCss = percent + '%';
    $('#js-load-amount').text(percent);
    $('.load-bar').css('height', percentCss);
  }
});
</script>

No JavaScript Fallback

If you'd like to support a fallback for users that aren't running JavaScript, you can add the original element in a noscript tag alongside the ProgressoJS version

<img class="js-progresso" data-src="img/example.jpg">
<noscript>
    <img src="img/example.jpg">
</noscript>

Why ProgressoJS?

ProgressoJS is one of the only accurate and declarative asset preloaders, which provides you with the tools you need to create your own loading screen animations. There are other asset preloaders or load animators, but most of them will require you to declare your preloaded assets in the head of your HTML document or may only simulate progress. ProgressoJS lets you declare your preloaded assets in their natural position in the HTML document, which means that it works with dynamic content and CMS's like WordPress, Drupal, ExpressionEngine, and Craft. ProgressoJS also taps into the actual sizes of your assets in order to provide you with an accurate progress measure that you can use for CSS or JavaScript animations.
